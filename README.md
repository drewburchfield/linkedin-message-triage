## What it does

Systematically reviews your LinkedIn inbox, separating real connections from solicitations. Identifies messages that need responses and drafts personalized replies with career context, including handling delayed response acknowledgments.

## Features

- Filter real connections from solicitations
- Identify messages needing responses
- Draft personalized replies with career context
- Handle delayed response acknowledgments

## Requirements

- [Playwright MCP](https://github.com/anthropics/anthropic-quickstarts/tree/main/mcp-playwright) server

## Install

```
claude plugins install linkedin-message-triage@not-my-job
```
